# FRET Guitar Trainer

เว็บฝึกกีตาร์แบบเรียลไทม์ สร้างด้วย Nuxt 4 และ Web Audio API

## เริ่มใช้งาน

```bash
npm install
npm run dev
```

เปิด `http://localhost:3000`

## คำสั่ง

- `npm run dev` — เปิดโหมดพัฒนา
- `npm run build` — สร้าง production build
- `npm run preview` — ทดลอง production build
- `npm run generate` — สร้าง static output
