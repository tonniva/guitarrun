10-minute routine

Learn notes in useful layers.

Accuracy comes first. Name each note aloud before speed becomes the goal.

01

Anchor the open strings

Say E, A, D, G, B, E from the lowest string to the highest. Then say them in reverse.

EADGBE
Pass when: You can name all six open strings in both directions without hesitation.
02

Map one natural note at a time

Choose C. Find every C from the open position through fret 12, then repeat with G and A.

Practice: Use GuitarRun to play a target note and verify every matching location visually.
03

Use random recall

Pick a string and a natural note. Find it before counting to three. Change strings after each answer.

Pass when: You can find 20 random natural notes with at least 90% accuracy.
Common questions

Guitar note trainer FAQ

What notes should I learn first?

Begin with the natural notes A, B, C, D, E, F, and G. Add sharps and flats after the natural-note landmarks feel familiar.

Should I memorize every note at once?

No. Learn one or two notes across all strings, use them in a scale or chord, and add another note only when recall becomes reliable.

Why say note names aloud?

Speaking forces active recall. It connects the physical location to a name instead of letting your fingers move through a familiar shape automatically.
