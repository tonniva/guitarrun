Five-minute setup

Connect, calibrate, and test.

Use a clean sound while setting up. Effects can create extra frequencies that make pitch detection less stable.

01

Connect the guitar

Interface: Plug the guitar into the instrument or Hi-Z input, then connect the interface to the computer by USB. Keep the input gain below clipping.

Target: The interface meter should move clearly when you play but never remain red.
02

Choose a clean input

In GuitarRun, press Connect, allow microphone access, and choose the interface input. If using the notebook microphone, place an acoustic guitar 20–40 cm away and point the sound hole slightly past the microphone rather than directly into it.

Avoid: Distortion, reverb, delay, speaker playback, fans, conversation, and other instruments.
03

Run Calibrate Input

Play the requested note at your normal practice volume. Calibration measures the useful signal level and separates your playing from the room noise.

Retest when: You change guitars, input gain, microphone position, room, or audio device.
04

Confirm stable notes

Play one string at a time and let it ring. The detected note should settle quickly instead of jumping between several names.

Ready for Marathon: Five slow notes are detected correctly without false hits between them.
Troubleshooting

When detection feels inaccurate

The note name keeps jumping

Lower distortion and gain, mute unused strings, move away from speakers, and play a single note cleanly. With a microphone, reduce room noise and move closer to the instrument.

Low strings respond slowly

Pick closer to the bridge for a clearer attack, raise the interface input slightly without clipping, and avoid heavy palm muting during calibration.

GuitarRun detects notes when I am not playing

The noise floor is too high. Turn off fans or speakers, reduce interface gain, use a quieter input, and run calibration again.

There is a noticeable delay

Select the interface directly instead of a virtual audio device, close other audio-heavy applications, and use a wired USB connection where possible.
